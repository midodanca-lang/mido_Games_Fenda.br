# Mido Fenda Studio - Projeto do site

Este é o projeto completo do site **Mido Fenda Studio**, pronto para importar no SPCK Editor ou outro editor Android e publicar no GitHub Pages.

## Estrutura
```
MidoGames_Project/
 ┣ index.html
 ┣ download.html
 ┣ store.html
 ┣ about.html
 ┣ media.html
 ┣ settings.html
 ┣ yourgame.html
 ┣ css/style.css
 ┣ js/script.js
 ┗ apk/ (coloque seus apks aqui)
```

## Como usar no Android (SPCK Editor)
1. Baixe o arquivo ZIP deste projeto e descompacte em uma pasta no seu celular.
2. Abra o SPCK Editor e importe a pasta do projeto.
3. Edite arquivos se quiser e visualize no navegador embutido.

## Como publicar no GitHub Pages
1. Crie um repositório no GitHub (ex: `mido_Games_Fenda.br`).
2. Faça upload de todos os arquivos e pastas (arraste e solte).
3. No repositório, vá em **Settings → Pages**, escolha a branch `main` e a pasta `/ (root)` e salve.
4. Acesse `https://<seu-usuario>.github.io/<seu-repositorio>/`

## Observações importantes
- Para uploads diretos de APK no site seriam necessários backend + armazenamento (posso ajudar a configurar se quiser).
- Por enquanto, os APKs podem ser hospedados no Drive/MediaFire/Dropbox e o link é colado ao publicar na página 'Publicar seu jogo'.
